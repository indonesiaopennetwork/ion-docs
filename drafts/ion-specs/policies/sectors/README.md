# `policies/sectors/` — sector-specific policies

This folder holds ION's policies whose terms apply within one sector
only. Cross-sector policies live in `../cross-sector/`.

## One folder per sector

```
policies/sectors/
├── trade/        ← Trade & Commerce policies
├── logistics/    ← Logistics policies
├── mobility/     ← Mobility policies
├── hospitality/  ← Hospitality policies
├── finance/      ← Finance policies
└── services/     ← Services policies
```

Each sector folder will contain policy categories specific to that
sector — return and warranty for Trade, evidence and insurance for
Logistics, no-show for Hospitality, and so on.

## What is here today

The folder structure is in place; sector policy terms documents are
not yet drafted.

See each sector's folder for what is planned there, and
[`policies/README.md`](../README.md) for the policy IRI mechanism.
