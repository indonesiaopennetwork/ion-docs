# `schema/core/identity/v1/` — placeholder

**Pack purpose:** Indonesian business and individual identity fields — NPWP, NIB, NIK, business form (PT/CV/UD/Perorangan).

**Mounts onto Beckn bag(s):** Provider.providerAttributes, Participant.participantAttributes.

## What this pack will contain

When drafted, this pack will hold:

- `attributes.yaml` — OpenAPI/JSON Schema definitions for the fields this pack contributes
- `context.jsonld` — JSON-LD context with the packs vocabulary URLs
- `vocab.jsonld` — linked-data vocabulary definitions
- `examples/` — sample payloads showing how this pack composes inside the target Beckn bag (governed zone, with empty `participantExtensions` and `ionExperimental` zones illustrated)

## What is here today

Empty. Schema files will be added once the relevant working group drafts them.

See [`schema/core/README.md`](../../README.md) for the cross-sector pack layer overview, and the top-level [`README.md`](../../../../README.md) for IONs overall extension model.
